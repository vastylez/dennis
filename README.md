# SimplyDennis Proposal Web

Single-page proposal website generated from:
`SimplyDennis - Digital Marketing Proposal.pdf`

## Run locally

```bash
cd simplydennis-proposal-web
python -m http.server 8080
```

Open: http://localhost:8080

## Deploy to Vercel (quick)

1. Push this folder to a GitHub repo.
2. In Vercel: **New Project** → import repo.
3. Framework preset: **Other** (static) is enough.
4. Deploy.

Alternative with Vercel CLI:

```bash
npm i -g vercel
vercel
```

## Notes

- Style follows your current proposal aesthetic: dark, premium, cyan accents.
- Content mirrors the key points from the provided PDF.
